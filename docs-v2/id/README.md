# Dokumentasi Lost & Found Platform

Selamat datang di pusat dokumentasi untuk **Lost & Found Platform** (PID154 Final Year Project). Dokumentasi ini disusun berdasarkan standar industri modern untuk memfasilitasi pengembangan lanjutan, audit, pengujian, dan operasional proyek.

## 📚 Daftar Isi

Dokumentasi ini dibagi menjadi beberapa modul spesifik untuk kemudahan navigasi:

1. **[Getting Started (Persiapan Awal)](./01-getting-started.md)**
   Panduan instalasi lokal, prasyarat sistem, dan konfigurasi environment variables (`.env`).

2. **[Architecture (Arsitektur Sistem)](./02-architecture.md)**
   Desain sistem secara keseluruhan, *tech stack*, struktur folder aplikasi, dan skema database (MongoDB).

3. **[API Reference (Referensi API)](./03-api-reference.md)**
   Dokumentasi *endpoint* internal Next.js, standar *payload* permintaan, autentikasi (JWT), dan format respons.

4. **[Core Features (Fitur Inti)](./04-core-features.md)**
   Penjelasan teknis tentang alur kerja fitur-fitur utama seperti QR Code Generation, *Realtime Notifications* (Ably), dan *Email System* (Resend).

5. **[Deployment & Operations (Deployment & Operasional)](./05-deployment.md)**
   Panduan *deployment* ke Vercel, *best practices* operasional, pemantauan (Analytics & Speed Insights), dan pemeliharaan aplikasi.
